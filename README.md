# Exercise-03b-Tweening-and-Screen-Shake

Exercise for MSCH-C220, 23 September 2021

Adding several "juicy" features to a simple match-3 game.

## To play

Select and drag tiles using the mouse.


## Implementation

Built using Godot 3.3.3
